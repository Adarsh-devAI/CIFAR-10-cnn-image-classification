# Image Classification using Deep CNN on CIFAR-10

This project implements a Deep Convolutional Neural Network (CNN) trained from scratch on CIFAR-10 dataset and deployed using a Gradio web interface.

## Features
- Deep CNN model
- CIFAR-10 dataset training
- Gradio based user interface
- Real-time image prediction

## Dataset
CIFAR-10 (60,000 images, 10 classes)

## Installation
pip install -r requirements.txt

## Run Training
python training/train.py

## Run Interface
python interface/app.py
